# JointQuant Framework

基于聚宽API文档搭建的回测和模拟运行策略的框架

数据源使用akshare

推送使用bark

## Installation Steps


```shell
$ git clone https://github.com/caolingyu/JoinQuant-Framework.git
$ cd JoinQuant-Framework/
$ pip install -r requirements.txt
```

## Usage Instructions

### 回测
```
python backtest.py
```
### 实时信号生成
```
python daily_operation.py
```
